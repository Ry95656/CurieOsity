# CurieOsity

## Project Overview
CurieOsity is an innovative platform designed to explore and ignite curiosity through interactive learning experiences and engaging content. Our mission is to make learning accessible, enjoyable, and effective for everyone.

## Features
- **Interactive Learning Modules**: Dive into a wide range of subjects with interactive content that promotes hands-on learning.
- **Engaging Community**: Connect with like-minded individuals and share insights, resources, and experiences.
- **Personalized Learning Paths**: Tailored suggestions based on user preferences and learning pace.
- **Progress Tracking**: Keep track of your learning journey and achievements over time.

## Tech Stack
- **Frontend**: React, Redux, Tailwind CSS
- **Backend**: Node.js, Express
- **Database**: MongoDB
- **Deployment**: AWS, Docker

## Setup Instructions
1. **Clone the repository**:
   ```bash
   git clone https://github.com/Ry95656/CurieOsity.git
   ```
2. **Navigate to the project directory**:
   ```bash
   cd CurieOsity
   ```
3. **Install dependencies**:
   ```bash
   npm install
   ```
4. **Start the development server**:
   ```bash
   npm start
   ```
5. **Open your browser** and navigate to `http://localhost:3000` to view the application.

6. **For production build**:
   ```bash
   npm run build
   ```
7. **Follow deployment instructions** in the [docs](https://link-to-deployment-docs.com).
